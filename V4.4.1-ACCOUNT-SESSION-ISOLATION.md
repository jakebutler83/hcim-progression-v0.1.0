# v4.4.1 Account Session Isolation

- Clears all in-memory group state whenever Firebase authentication changes.
- Stops old Firestore progress, activity, and live-location listeners on logout or account switching.
- Validates the saved group against the signed-in UID before opening it.
- Clears invalid group references from the user profile instead of displaying another account's group.
- Reinitializes local progress state for every group-ready event.
- Scopes live-map icon preferences to the signed-in account.
- Updates the service-worker cache to v4.4.1.
