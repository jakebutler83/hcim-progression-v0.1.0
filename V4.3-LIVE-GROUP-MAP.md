# HCIM Progression v4.3 — Live Group Map

This release adds a real-time Firestore-powered group map.

What it does:
- Listens to groups/{groupId}/locations in real time.
- Shows online/offline status using a 45-second freshness window.
- Places markers from RuneLite X/Y world coordinates.
- Displays world, region ID, coordinates, plane, and last update.
- Uses a stylized Gielinor coordinate map with no external map dependency.

Known limitation:
- Instanced, underground, and separate-plane areas currently use their raw template/world coordinates.
- A future tile-map update can replace the stylized map without changing the Firestore data model.
