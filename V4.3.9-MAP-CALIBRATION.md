# v4.3.9 — Sailing map coordinate calibration

The Sailing-era official overview image includes margins/cropping that do not share the exact origin used by RuneLite world coordinates.

This release applies a surface-map display calibration of:

- X: +145 game tiles
- Y: -70 game tiles

The Firestore/RuneLite coordinates are not changed. Only their position on the overview image is corrected. The calibration was based on the Port Sarim, Rimmington and Crandor mainland area.
