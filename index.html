🟢 Live Sync Setup

Checking Firebase setup...

  1. Create a Firebase project.
  2. Create a Firestore Database in test mode.
  3. Open firebase-config.js in VS Code.
  4. Paste your Firebase config into that file.
  5. Upload the whole folder to Netlify.
  6. Share the Netlify link with your friends.

Important: Test mode is okay for a tiny private friend tracker, but anyone with the link could change data. Use a random Netlify URL or add login later.
