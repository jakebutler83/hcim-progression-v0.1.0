# v4.3.13 — Regional and Interior Map Layers

- Keeps the normal surface world inside a clean surface-only boundary.
- Prevents distant dungeon, instance, and hidden map chunks from appearing beside the overworld when zoomed out.
- Automatically changes to a local dungeon/interior map when the selected player enters coordinates outside the main surface canvas.
- Uses the player's RuneLite plane (0–3) for the tile layer, so upstairs and downstairs floors load separately.
- Loads Prifddinas/Tirannwn as its own regional map because it occupies a separate coordinate canvas.
- Shows city labels only for the active regional map.
- Reset Map now centers the active region rather than jumping to the top-left or another map canvas.
