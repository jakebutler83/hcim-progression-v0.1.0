# v4.3.16 Negative Zoom Tile Fix

- Adds `minNativeZoom: 0` to the OSRS Leaflet tile layer.
- Display zooms -1 through -4 now scale native zoom-0 map tiles instead of requesting nonexistent negative zoom directories.
- Keeps the wider zoom-out range while preventing the gray-map failure.
- Adds a small tile error diagnostic hook and retains nearby tiles during panning.
