# v4.3.4 Live Map Fix

- Removed the separate **View Varlamore** selector. Varlamore is part of the current RuneScape Wiki map itself.
- The selected teammate now automatically refreshes on every Firestore location update.
- Added live-follow and explore states.
- While live-follow is active, the map recenters whenever the player moves.
- When the embedded map is manually explored, the screen-centered overlay is hidden so it cannot falsely appear attached to the wrong coordinate.
- **Resume live follow** returns to the selected player.
- Bumped the service-worker cache so Netlify clients receive the new JavaScript and HTML.

The RuneLite companion already sends a location update approximately every five game ticks, so no plugin code change was required for automatic website updates.
