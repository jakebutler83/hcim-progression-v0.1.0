# HCIM Progression v4.2 — Companion Linking

1. Copy this website package into the website master Git folder.
2. Deploy to Netlify.
3. Confirm the existing Firebase Admin environment variables are still set.
4. Merge and publish FIRESTORE-RULES-COMPANION.txt into your current rules.
5. Open Website Settings and generate a one-time link code.

The link code expires in 10 minutes and may only be used once.
The RuneLite device token is stored locally in RuneLite configuration.
Firebase Admin credentials remain server-side in Netlify.
