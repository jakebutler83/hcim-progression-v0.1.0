# v4.3.3 — Player Marker and Varlamore Map Controls

Fixes:
- Removes the unsupported map-version query parameter.
- Adds a clearly visible player marker at the center of the embedded map.
- The marker updates with the selected teammate's name, world, and coordinates.
- Adds a View Varlamore button.
- Adds a Reload Map button to force the embedded Wiki map to reload.

Important:
The centered marker is accurate because selecting a teammate centers the map on
that teammate's RuneLite X/Y/plane coordinate.
