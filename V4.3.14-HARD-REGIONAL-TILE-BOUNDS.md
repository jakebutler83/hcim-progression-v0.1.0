# v4.3.14 — Hard Regional Tile Bounds

- Applies the active region bounds directly to the Leaflet tile layer, not only the camera.
- Recreates the tile layer when switching surface, Prifddinas, dungeon, or floor contexts.
- Prevents unrelated coordinate chunks from being requested or rendered around the active map.
- Only displays player markers belonging to the active regional map and plane.
- Uses full max-bounds viscosity to keep navigation inside the active region.
