# v4.3.11 Coordinate Map Fix

- Replaced the decorative overview-image projection with coordinate-accurate OSRS map tiles.
- RuneLite x/y coordinates now map directly to Leaflet CRS.Simple coordinates.
- Keeps the last valid surface position while a player is in an underground coordinate band.
- Plane changes no longer rebuild or reset the map.
- Reload Map redraws tiles and returns the camera to the center of Gielinor instead of the top-left.
- Zoom range is -4 through 8, allowing a much farther zoom-out.
