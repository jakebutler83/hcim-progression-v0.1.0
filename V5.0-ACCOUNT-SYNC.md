# v5.0 Account Sync

Adds `companion-account-sync`, an authenticated Netlify Function that stores a RuneLite account snapshot and updates completed quest rows plus supported skill-threshold tasks for the linked member slot.

The RuneLite companion adds **Sync Account Now**. It uploads current skill levels/XP and completed quests. Manual inventory, bank, minigame, and subjective checklist tasks remain manual. Achievement diary bulk sync is not included in this first release because RuneLite does not expose one universal diary-completion API.
