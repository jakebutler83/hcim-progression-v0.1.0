# HCIM Progression v4.3.2 — Varlamore Map Refresh

The embedded OSRS Wiki map is a live hosted map and already includes current
Varlamore content. This patch adds a cache-busting version parameter and bumps
the HCIM service-worker cache so browsers stop reusing the older embedded view.

After deployment:
1. Press Ctrl+F5.
2. If needed, unregister the old service worker once.
3. Reopen Live Map.
