# v0.4.2 — Achievement Diary Region Artwork

This release implements the diary artwork directly in code.

## Included
- CSS-only panoramic banners for every diary region
- Distinct regional palettes and landmark silhouettes
- Per-region completion rings
- Member-task completion percentage
- Animated banner hover and regional progress bar
- No external screenshots or copyrighted game artwork required
- Responsive mobile layout and reduced-motion support

## Test
1. Open Achievement Diaries.
2. Verify every region has a banner.
3. Check tasks and confirm the regional percentage updates.
4. Search/filter and verify banners remain attached to the correct region.
5. Test mobile width and each interface theme.
