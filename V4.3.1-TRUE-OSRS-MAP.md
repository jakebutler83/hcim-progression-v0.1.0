# HCIM Progression v4.3.1 — True OSRS Map

This release replaces the fictional CSS map with the actual interactive OSRS Wiki map.

Features:
- Real Old School RuneScape terrain and world-map layers.
- Zooming and panning through the embedded Doogle Maps interface.
- Real-time Firestore teammate list.
- Click a teammate to center the real map on their RuneLite X/Y/plane coordinates.
- Open any teammate's position in the full map in a new tab.
- Online/offline freshness remains 45 seconds.

Technical note:
Because the Wiki map is hosted on another origin, custom website markers cannot be drawn inside the embedded map frame. This release centers the real map on the selected teammate. A future self-hosted tile layer can support simultaneous custom markers directly over the terrain.
